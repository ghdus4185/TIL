- 도메인 : 사람이 보기에 복잡한 주소 대신 쓰는 가짜 주소

## HTML
- Web의 뼈대

- W3C - 웹 표준 사이트, 모르는게 있으면 여기서 찾아본다.

get 요청 : 서버로 부터 문서파일 하나를 받음
post 요청: 클라이언트가 보낸 정보를 받아서 서버에 저장
